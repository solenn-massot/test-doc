## \[GET\] Artiste/\{id_artiste\}

### Description

Récupérer les informations d'un artiste.

### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| id_artiste      | int | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| id      | int |  
| nom    |   string    |
| id_mouvement     |   int    |
| anecdote     |   string    |
| img     |   string    |
