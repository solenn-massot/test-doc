## \[GET\] Oeuvre/mouvement\{id_mouvement\}

### Description

Récupérer toutes les oeuvres d'un mouvement.

### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| id_mouvement     | int | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| id      | int |  
| titre    |   string    |
| date    |   int    |
| id_mouvement     |   int    |
| id_artiste     |   int    |
| anecdote     |   string    |
| img     |   string    |
