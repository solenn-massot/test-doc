## \[POST\] Utilisateur/

### Description

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| identifiant      | string | obligatoire |
| email      | string | obligatoire |
| mdp      | string | obligatoire |

### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| status      | bool |  
| message     |   string    |
