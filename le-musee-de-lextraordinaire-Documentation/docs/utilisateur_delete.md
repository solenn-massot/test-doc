## \[DELETE\] Utilisateur/\{id_utilisateur\}

### Description

Supprimer le compte et les informations associés de l'utilisateur connecté.
Cette fonction permet de supprimer uniquement son propre compte.


### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| id      | int | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| status      | bool |  
| message     |   string    |
