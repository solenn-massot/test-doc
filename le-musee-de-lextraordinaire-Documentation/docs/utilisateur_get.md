## \[GET\] Utilisateur/\{id_utilisateur\}

### Description

Récupérer les informations de l'utilisateur connecté.
Cette fonction permet de récupérer uniquement ses propres informations.


### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| id      | int | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| id      | int |  
| identifiant      |   string    |
| email |   string    |
